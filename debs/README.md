# Place your .deb files here

Drop your `.deb` packages in this folder, then commit and push.

GitHub Actions will automatically generate the `Packages` and `Packages.bz2` files for you.

## Example:
```
debs/
├── com.example.tweak_1.0.0_iphoneos-arm.deb
├── com.example.app_2.1.0_iphoneos-arm64.deb
└── README.md (this file)
```
